# DoseTrack 💊

**Predict how your meds will make you feel.**

A drug effects prediction and tracking Progressive Web App built with Next.js, TypeScript, Tailwind CSS, and Recharts.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Disclaimer

DoseTrack is not medical advice. Predictions are estimates based on published pharmacokinetic data and user self-reports.
